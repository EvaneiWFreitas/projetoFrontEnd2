🌐 Projeto Web com HTML, Bootstrap e CSS

Este projeto consiste na criação de um site simples utilizando HTML, Bootstrap e CSS, com o objetivo de praticar conceitos fundamentais de desenvolvimento web e construção de interfaces responsivas.

🛠️ Tecnologias utilizadas
🌐 HTML5
🎨 CSS3
📦 Bootstrap
📌 Sobre o projeto

O site foi desenvolvido com foco na estruturação de páginas web utilizando HTML, aliado ao uso do Bootstrap para facilitar a criação de layouts responsivos e modernos. O CSS foi aplicado para personalizações visuais, garantindo uma identidade própria ao projeto.

Durante o desenvolvimento, foram aplicados conceitos como organização de layout, utilização de componentes prontos do Bootstrap e adaptação para diferentes tamanhos de tela.

✨ Funcionalidades
📄 Estrutura básica de um site
📱 Layout responsivo com Bootstrap
🎨 Estilização personalizada com CSS
🧭 Navegação simples entre seções
📱 Responsividade

O projeto foi desenvolvido utilizando Bootstrap, garantindo que o site se adapte a diferentes dispositivos, como celulares, tablets e desktops.

🎯 Objetivo
Praticar o uso de HTML, CSS e Bootstrap
Entender a construção de layouts responsivos
Desenvolver um site simples do zero
Evoluir habilidades em front-end
